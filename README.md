# Claude Othello

ブラウザで動作するシンプルなオセロ（リバーシ）ゲームです。
外部ライブラリ不要・ビルド不要で、GitHub Pages に直接デプロイできます。

---

## プレイ方法

1. 白いドットが表示されているマスをクリックして石を置く
2. CPU が自動的に手を打つ
3. どちらも置けなくなった時点でゲーム終了、石の多い方が勝ち
4. 「もう一度プレイ」ボタンで再スタート

---

## 機能仕様

### ゲームルール

| 項目 | 内容 |
|------|------|
| ボードサイズ | 8×8 |
| プレイヤー | 黒（あなた） vs 白（CPU） |
| 先手 | プレイヤー（黒） |
| 初期配置 | 標準オセロ配置（中央 2×2） |
| 石の反転 | 縦・横・斜め 8 方向すべて対応 |
| パス | 置ける場所がない場合は自動でパス |

### UI・表示

- 有効な手をドットで強調表示（ホバーでハイライト）
- スコアボードにリアルタイムで石数を表示
- 現在の手番を日本語で表示
  - `あなたの番` / `CPUが考え中…` / `ゲーム終了`
- パス・勝敗結果をメッセージで通知

### アニメーション

| アニメーション | 内容 |
|------|------|
| 石を置く | ポップイン（scale 0 → 1、0.22 秒） |
| 石が反転する | フリップエフェクト（scale 縮小→拡大、0.28 秒）＋段階的遅延 |
| アクティブプレイヤー | 黄色グロー表示 |

### AI 仕様

CPU は以下のハイブリッド戦略で手を選択します。

- **20%** : ランダムな有効手を選択
- **80%** : 評価関数により最善手を選択

#### 評価関数の内訳

| 要素 | 内容 |
|------|------|
| 位置評価 | コーナー +100、辺 +5〜+10、X マス −50 など |
| 機動力 | 有効手の差 ×10 点 |
| 石数差 | 石数の差（終盤は重み 3 倍） |

CPU は手を打つ前に 550〜1050 ms のランダム待機を挟みます。

## ローカルでの起動方法

```bash
# Python 3
python3 -m http.server 8000
```

起動後、ブラウザで `http://localhost:8000` を開いてください。

> `index.html` をブラウザで直接開いても動作しますが、HTTP サーバー経由を推奨します。

- ES6・CSS Grid・CSS アニメーション対応が必要
