# Claude Othello

ブラウザで動作するシンプルなオセロ（リバーシ）ゲームです。
外部ライブラリ不要・ビルド不要で、GitHub Pages に直接デプロイできます。

---

## プレイ方法

1. ゲーム起動時に **対戦相手（難易度）** を選択する
2. 白いドットが表示されているマスをクリックして石を置く
3. CPU が自動的に手を打つ
4. どちらも置けなくなった時点でゲーム終了、石の多い方が勝ち
5. 「もう一度」ボタンで同じ相手と再戦、「相手を変える」ボタンで選択画面に戻る

## 機能仕様

### ゲームルール

| 項目 | 内容 |
|------|------|
| ボードサイズ | 8×8 |
| プレイヤー | 黒（あなた） vs 白（CPU） |
| 先手 | プレイヤー（黒） |
| 初期配置 | 標準オセロ配置（中央 2×2） |
| 石の反転 | 縦・横・斜め 8 方向すべて対応 |
| パス | 置ける場所がない場合は自動でパス |

### 対戦相手・難易度

ゲーム開始時に 3 種類の CPU から対戦相手を選択できます。

| 難易度 | キャラクター | 見た目 | AI 戦略 |
|--------|-------------|--------|---------|
| 弱 ★☆☆☆☆ | スライム | 緑のスライム（CSS アート） | 85% ランダム / 15% 評価関数 Greedy |
| 中 ★★★☆☆ | ナイト | 青いヘルメット（CSS アート） | 20% ランダム / 80% 評価関数 Greedy |
| 強 ★★★★★ | 魔王 | 赤い角と光る目（CSS アート） | Minimax + α-β 枝刈り（深さ 4） |

### UI・表示

- 有効な手をドットで強調表示（ホバーでハイライト）
- スコアボードにリアルタイムで石数を表示
- 現在の手番を日本語で表示
  - `あなたの番` / `CPUが考え中…` / `ゲーム終了`
- パス・勝敗結果をメッセージで通知
- スコアボードの CPU 名が選択した難易度名に切り替わる

### キャラクター表情

ゲーム中、ボードの上に選択した対戦相手のキャラクターが表示されます。スコア差に応じてリアルタイムで表情とセリフが変化します。

| 状況 | スライム | ナイト | 魔王 |
|------|---------|--------|------|
| CPU が優勢（+4 以上） | 目が細くにっこり笑顔 | バイザーが青く光る | 目が激しく輝き、口が広がる |
| 接戦（±3 以内） | 口が一文字 | バイザーが暗い | デフォルト表情 |
| CPU が劣勢（−4 以下） | 目が垂れ、口が逆笑い | バイザーが赤く光る | 目が暗く、口が小さく縮む |

セリフも難易度ごとに個性があります（例：スライム `うわーん！` / ナイト `くっ…` / 魔王 `な、なんと…`）。ゲーム終了時は結果に合わせた専用セリフに切り替わります。

### アニメーション

| アニメーション | 内容 |
|------|------|
| 石を置く | ポップイン（scale 0 → 1、0.22 秒） |
| 石が反転する | フリップエフェクト（scale 縮小→拡大、0.28 秒）＋段階的遅延 |
| アクティブプレイヤー | 黄色グロー表示 |

### AI 仕様

#### 弱（スライム）

- 85% の確率でランダムな有効手を選択
- 15% の確率で評価関数による最善手を選択

#### 中（ナイト）

- 20% の確率でランダムな有効手を選択
- 80% の確率で評価関数による最善手を選択

#### 強（魔王）

- **Minimax アルゴリズム**（深さ 4）＋ **α-β 枝刈り** による先読み

#### 評価関数の内訳（中・強 共通）

| 要素 | 内容 |
|------|------|
| 位置評価 | コーナー +100、辺 +5〜+10、X マス −50 など |
| 機動力 | 有効手の差 ×10 点 |
| 石数差 | 石数の差（終盤は重み 3 倍） |

CPU は手を打つ前にランダム待機を挟みます（弱・中: 550〜1050 ms、強: 800〜1400 ms）。

## ローカルでの起動方法

```bash
# Python 3
python3 -m http.server 8000
```

起動後、ブラウザで `http://localhost:8000` を開いてください。

> `index.html` をブラウザで直接開いても動作しますが、HTTP サーバー経由を推奨します。

## 動作環境

- モダンブラウザ（Chrome / Firefox / Safari / Edge）
- ES6・CSS Grid・CSS アニメーション対応が必要
- モバイル対応（360px〜、タッチ操作可）
